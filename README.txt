== Description ==

Alt Text Automator automatically generates alt text for your images.

**AI-powered:** Whenever you upload an image, our AI will analyze it and add descriptive alt text to the image.

**Multiple Languages:** Over 130 languages for alternative text.

**Bulk Actions:** Bulk generate alt texts for images in your media library or posts.

**Try for FREE:** A free trial is available for new users.
